This file contains an example of bitwise operations in C.

There are 6 operands used to perform bitwise operations in C:
- ~ is used to compliment all bits
- | is used to set a bit
- & is used to clear a bit
- ^ is used to toggle a bit
- << and >> are used to rotate a variable n times arithmetically
