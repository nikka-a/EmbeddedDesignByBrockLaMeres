The file contains an example of Symbolic Addressing Mode. In this mode, the MSP430 allows use of address labels, a unique name associated with an address location, as operands in both ```src``` and ```dst```. No special syntax is required to use it. Memory is allocated the same way as in [04_ADDRMODE3_ABSOLUTE](./asm/04_ADDRMODE3_ABSOLUTE) exercise.

Concepts covered:
- data movement using symbolic addressing mode
