This file contains an example of an overflow-based conditional jump. ```jge``` instruction will jump when N and V has different values (1 or 0). ```jl``` instruction will jump when N and V flags have the same values. See more [here](/asm/20_FLOW_JUMP_N_BRANCH) and [here](/asm/21_FLOW_CARRY_JMP).
