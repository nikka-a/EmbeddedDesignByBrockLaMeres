The file contains example of Register Mode Addressing. In this mode, the registers are simply used in operands. It can be used in both ``` src ```, and ```dst```. This means that there is no memory address usage and the data is moved only between the registers. It also includes difference between a word and a byte: ```.w``` denotes a 16 bits word, ```.b``` denotes 8 bits word.

Concepts covered:

- memory movement between registers
