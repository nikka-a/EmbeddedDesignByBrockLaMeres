This file contains an example of bitwise operations ```bis``` (bit set) and ```bic``` (bit clear). ```bis``` will set the bits in ```dst``` correspoding to the 1s in the ```src``` mask. ```bic``` will clear the bits in ```dst``` corresponding to the 1s in the ```src``` mask. In these instructions, 0s will leave the ```dst``` unanaltered. 

Concepts covered:
- bitwise operations ```bis``` and ```bic```.
