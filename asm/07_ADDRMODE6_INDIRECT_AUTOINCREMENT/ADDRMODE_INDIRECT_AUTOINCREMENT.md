The file contains an example of Indirect Autoincrement Addressing Mode. This mode is similar to [Indirect Register Addressing Mode](./asm/06_ADDRMODE5_INDIRECTREGISTER), however an ```+``` is added that indicates that after using the address, it is incremented by 2 if a ```.w``` is used, or 1 when a ```.b``` is used.

Concepts covered:
- data movement using indirect autoincrement mode
