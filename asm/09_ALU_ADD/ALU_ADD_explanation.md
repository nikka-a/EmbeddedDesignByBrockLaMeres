This file contains an example of an arithmetic operation ```add``` performed by the ALU of MSP430. It demonstrates how the MCU handles overflow via Carry Bit (C) and Overflow Bit (V). C flag is set when the result of a word operation produces a carry and is cleared when no carry occured. V flag is set when the result of an arithmetic operation overflows the signed-value range.

Concepts covered:
- arithmetic operations (signed and unsigned)
- Carry (C) and V (Overflow) flags
